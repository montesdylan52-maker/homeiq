# HomeIQ — Inmobiliaria Digital

Plataforma web inmobiliaria para visualizar y cotizar departamentos en Lima. Desarrollada con React, Node.js y SQLite.

## 🚀 Tecnologías

- **Frontend:** React, React Router DOM, Lucide React
- **Backend:** Node.js, Express
- **Base de datos:** SQLite (better-sqlite3)
- **Estilos:** CSS personalizado con variables y diseño responsivo

## ✨ Funcionalidades

- Carrusel de imágenes interactivo con flechas y auto-play
- Listado de departamentos con filtros por distrito
- Página de detalle por departamento
- Formulario de cotización conectado a base de datos
- Navbar responsivo con scroll detection
- Preguntas frecuentes con acordeón
- Página Nosotros

## ▶️ Cómo correr el proyecto

### Backend
```bash
cd server
node index.js
```

### Frontend
```bash
npm start
```

El frontend corre en `localhost:3000` y el backend en `localhost:5000`.
